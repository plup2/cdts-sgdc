Leaving a secure site

  • This link https://www.google.ca has a hostname (google.ca) different from yours. You'll be redirect to exiturl-en.html page because the exitURL parameter contains the value exiturl-en. html.
  • This link https://www.google.ca will open in another window. Through the targetWarning parameter, you can set a message that will warn users that the link will open in another window. This message will append the existing display message.
  • This link breadcrumbs-en. html has the same hostname as yours. You won't be redirect to exiturl-en.html.
  • This link https://google.com has his hostname (google.com) in the exception list exitDomains. You won't be redirect to exiturl-en.html.
